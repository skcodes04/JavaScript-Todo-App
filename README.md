# 📝 JavaScript Todo App

A modern Todo App built with HTML, CSS, and JavaScript. This project allows users to add, delete, and save tasks using the browser's localStorage.

## ✨ Features

- Add new tasks
- Delete tasks
- Save tasks with localStorage
- Tasks remain after page refresh
- Add tasks by pressing Enter
- Responsive design
- Clean and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- localStorage API

## 📂 Project Structure

```
├── index.html
├── style.css
├── script.js
└── README.md
```

## 🚀 How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
